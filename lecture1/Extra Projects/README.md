# Calculator Hub

A multi-tool calculator application built in Python as part of my CS50P journey.

This project combines multiple calculators into one menu-driven terminal application and focuses heavily on:
- input validation
- modular programming
- user interaction
- problem-solving logic
- clean program structure

## Features

### 1. Basic Calculator
Supports:
- Addition
- Subtraction
- Multiplication
- Division

Includes:
- division-by-zero handling
- formatted outputs

---

### 2. Ohm’s Law Calculator
Calculates:
- Voltage (U)
- Resistance (R)
- Current (I)

Includes:
- validation for invalid electrical values
- unit-formatted outputs

---

### 3. BMI Calculator
Supports:
- Metric units
- Imperial units

Features:
- automatic unit conversion
- BMI classification:
  - Underweight
  - Healthy
  - Overweight
  - Obese
- detailed input validation

---

### 4. Temperature Converter
Supports conversions between:
- Celsius
- Fahrenheit
- Kelvin

Includes:
- absolute-zero validation
- multiple conversion paths

---
