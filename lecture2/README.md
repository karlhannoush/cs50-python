# Lecture 2 — Loops

This lecture focused on loops, string processing, and building more structured programs.

Compared to the previous lectures, the exercises here required more problem decomposition and more careful logic, especially for validating patterns and processing text step by step.

## Problem Set

### camel.py
Converts variable names from camelCase to snake_case.

Concepts practiced:
- for loops
- string iteration
- uppercase/lowercase handling
- string building

---

### coke.py
Simulates a Coke machine that accepts coins until enough money is inserted.

Concepts practiced:
- while loops
- input validation
- running totals
- loop control

---

### twttr.py
Removes vowels from user input.

Concepts practiced:
- loops
- conditionals
- string filtering

---

### plates.py
Checks whether a vanity plate is valid according to a set of rules.

Concepts practiced:
- breaking problems into helper functions
- validation logic
- loops and conditionals
- string analysis

This was one of the most logic-heavy exercises so far.

---

### nutrition.py
Displays the calories of fruits using a dictionary lookup.

Concepts practiced:
- dictionaries
- key-value access
- case normalization

## Extra Project — Text Analyzer Toolkit

This lecture’s extra project was a multi-feature text analysis program built entirely in Python.

Features:
- Character Counter
- Word Counter
- Sentence Counter
- Vowel Remover
- Palindrome Checker
- Frequency Analyzer

The goal of this project was to combine multiple concepts from the first three lectures into one structured program instead of solving isolated exercises.

Main concepts practiced:
- functions
- loops
- menus and program flow
- string processing
- conditionals
- decomposition into smaller subproblems

This project took significantly more effort than the lecture exercises and involved multiple redesigns and debugging sessions, especially for the sentence counter logic.
