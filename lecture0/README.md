# Lecture 0 — Functions and Variables

This folder contains my solutions for CS50P Lecture 0.

The lecture focused on:
- variables
- functions
- user input
- strings
- basic program structure

## Problem Set

### indoor.py
Converts user input to lowercase.

### playback.py
Replaces spaces with ellipses to simulate slower speech.

### faces.py
Converts text emoticons into emoji.

### einstein.py
Calculates energy using Einstein’s equation:

E = mc²

### tip.py
Calculates a restaurant tip based on meal cost and percentage.
