# CS50 Python Journey

This repository documents my progress through CS50’s Introduction to Programming with Python.

I started this course after finishing my first year in Computer and Communications Engineering (CCE) at USJ. I already had some prior exposure to Python from a university programming course, so some projects in this repository use concepts slightly beyond the current CS50 lecture, such as loops and more advanced program structure.

My goal is not just to finish the course, but to:
- build strong Python fundamentals
- improve problem-solving skills
- learn how to structure larger programs
- prepare for future AI and software engineering projects

## Repository Structure

- `lecture0/`
- `lecture1/`
- future lectures and projects

Each lecture folder contains:
- CS50 problem set solutions
- notes and experiments
- additional projects built to reinforce concepts

## Extra Projects

I also build extra projects outside the official problem sets to practice applying concepts in larger and more realistic programs.

These projects focus heavily on:
- debugging
- input validation
- modularity
- program organization
- practical problem solving
